## Useful commands

- Check the version of the Keptn CLI as well as the Keptn cluster (if authenticated)

  ```
  keptn version
  ```

- Check the status of your Keptn CLI

  ```
  keptn status
  ```

- Check what you can do with the Keptn CLI

  ```
  keptn --help
  ```
